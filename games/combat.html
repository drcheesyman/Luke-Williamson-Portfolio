Overview

Olympus Quest is an Action RPG inspired combat system. I created this project to practice creating VFX and feedback for player actions. This gave me an oportunity to develop new skills in creating shaders, animations, and code for a combat system. I also developed a new camera controller that follows the player dynamically, detects colision, and x-rays the player when they are behind an object.

This project is curently under development.
